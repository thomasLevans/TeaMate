SECTION *
-use the following format when writing your own test cases.
Note that each <> element of the test case is delimited by a new line.
Treat any text outside <> as constant.

FILENAME: testCase<testID>.txt
CONTENTS:
<testID>
<Requirement>
<module> *DO NOT INCLUDE MARTUS IN THE MODULE NAME*
<class name>
<method name>
<driver used>
[inputs...] *DELIMIT BY SPACES*
testOracle<testID>.txt
